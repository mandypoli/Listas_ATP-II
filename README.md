### 👩‍💻 Como rodar os códigos 
Para testar qualquer um dos exercícios, abra o terminal, navegue até a pasta da unidade correspondente usando o comando `cd` (Change Directory) e compile com o GCC. 

Exemplo para rodar o exercício 1 da Unidade 3:

```bash
# 1. Entre na pasta da Unidade
cd Unidade_3

# 2. Compile o arquivo
gcc 1.c -o programa

# 3. Execute o código
./programa
```
